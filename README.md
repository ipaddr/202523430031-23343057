# Tugas Minggu 2

## Ringkasan Materi & Implementasi

Pada minggu kedua, tugas yang telah dilakukan meliputi:

- Pembuatan halaman Verifikasi Email
- Pembuatan halaman utama Notes dengan fitur Logout
- Penanganan error dengan dialog
- Pengelolaan routing menggunakan named routes
- Setup awal penyimpanan CRUD
- Implementasi Unit Testing untuk menguji alur autentikasi

## Deskripsi Singkat

Pada minggu ini dilakukan perapian struktur kode aplikasi MyNotes, terutama pada bagian autentikasi. Kode dipisahkan ke dalam folder-folder yang lebih terstruktur seperti constants, enums, services, utilities, dan views. Ditambahkan juga halaman verifikasi email, halaman utama Notes dengan fitur logout, serta penanganan error yang lebih baik. Unit testing juga mulai diterapkan untuk memastikan alur autentikasi berjalan dengan benar.
